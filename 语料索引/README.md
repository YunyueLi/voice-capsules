# 语料索引 · 公开仓政策

本目录是**唯一**进入公开仓的语料相关物（正文 `原文/`、声部工作区 `声部/` 均 gitignore）。

## 政策（2026-07-07 收紧）
公开仓只列**同时满足**以下条件的作品/作者，且每行须带溯源与版权字段：
1. 原文法域 + 美国法域（仓库托管地）**双双公版**；
2. 作者名已消歧、确认语料非错配；
3. 附 来源 URL / 版权状态 / 获取日期。

**不列**版权期内/在世/URAA 恢复版权作者的任何「占有证据」（字数、文件数）——字数即持有全文的自证。
